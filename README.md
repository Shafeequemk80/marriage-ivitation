# Next.js Mongo Task Manager

Features:
- Next.js 14 App Router
- Tailwind CSS UI
- MongoDB + Mongoose
- Add / Edit / Delete entries
- Mark completed
- Search, sort, pagination
- Export to Excel & PDF
- Simple admin login (demo)

## Setup

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.local.example` to `.env.local` and set your MongoDB URI:

```env
MONGODB_URI=your_mongodb_connection_string
```

3. Run dev server:

```bash
npm run dev
```

4. Open http://localhost:3000
